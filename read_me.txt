# Personal Portfolio Website Project

This project is the final assignment for the HTML+CSS module, where we built a personal portfolio website to showcase the projects completed during the course.

## General Description
The website serves as a personal portfolio, designed to demonstrate your abilities in web design and development. It includes a gallery showcasing at least 6 projects, each with a representative image and description to highlight the tools and technologies used.

### Key Features
1. **Responsive Design**: The website is fully responsive, ensuring it works well on both desktop and mobile devices.
2. **Project Gallery**: A gallery displaying projects completed during the course, with images and descriptions for each project.
3. **Navigation Menu**: A sticky navigation bar at the top of the page allowing users to navigate between sections.
4. **Contact Form**: A form that allows visitors to submit their contact information.

## Website Structure
The website is divided into several sections:
1. **Header**: Includes a logo and navigation bar linking to different sections of the website.
2. **Introduction Section**: A welcoming message with a headline and introduction paragraph.
3. **About Section**: Contains personal information (including a photo or personal logo) and a link to download your resume.
4. **Gallery Section**: Displays all your projects, with images and descriptions for each one.
5. **Contact Section**: A contact form with fields for name, phone, email, and message.
6. **Footer**: Includes credit for the website's creation.

## How to Run the Project
1. Download the project files.
2. Open the `index.html` file in your web browser.

### Folder Structure
- `/index.html`: The main HTML file.
- `/css/`: Contains all stylesheets (CSS).
- `/images/`: Contains all images used in the website.
- `/projects/`: Contains project files and pages.

## Technologies Used
- **HTML5**: For structuring the content.
- **CSS3**: For styling the website.
- **Bootstrap / SASS**: For advanced styling (if used).
- **Git**: For version control and project submission.
